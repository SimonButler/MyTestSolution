For Database Updates run

dotnet ef database update -c #theCONTEXT#
i.e. dotnet ef database update -c SchoolContext
from cmd in project dir

To add a new migration run
dotnet ef migrations add #MIGRATIONDESCRIPTION# -c #theCONTEXT#
i.e. dotnet ef migrations add NewCourses -c SchoolContext


If you are using this project on a new computer you might nee to run a database update -c 

To run after configureing the project for ssl you will need to run without debugging (ctrl + F5).
This will allow the evnironment to generate the ssl certificate. After generating the SSL certificate you can the run with F5 like usual.
And if 500.19 error occurs saying it cannot read config file then you will need to restart VS and run as administrator.
